# Protect seller home addresses

Jualokal never automatically reveals or permanently unlocks a seller's home address. To preserve seller convenience, a seller may opt in per transaction to use their front gate as the exact Handover Point for a Trusted Buyer; the point is shown only after payment, exists only for that transaction, and never permits home entry or physical browsing.

The prototype may use a real Home Anchor for a consenting builder account only in protected private data. Real anchors never appear in source files, documentation, screenshots, seeded demo data, or the public repository; all Demo Sellers use fictional anchors. At launch, Home Anchors are encrypted, hidden from ordinary staff, accessible to specially authorized safety staff only when necessary with access recorded, shown to members only as approximate distance or area information, and deleted when no longer needed subject to legitimate retention requirements.
