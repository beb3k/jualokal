# Cap listing discovery at ten kilometres

Jualokal never shows a buyer listings more than 10 km from the buyer's current device location because a wider market would undermine its near-home purpose and recreate the travel burden it is designed to remove. The deliberately narrow prototype uses a 2 km Discovery Radius for a genuinely hyperlocal experience. Exact Home Anchors and Pilot Area boundaries remain hidden, and buyers see only rounded distance.

Viewing listings requires a Browsing Location while the app is in use. Jualokal uses it only to apply the Discovery Radius, never shows it to sellers, and does not retain browsing-location history. If location permission is denied or the location is unavailable, the member sees an explanation instead of listings; Demo accounts use a simulated location.
